# Project Planning Docs

This folder contains the lightweight planning artifacts for the activity tracker.

- `docs/requirements.md` - product goals, user stories, and scope.
- `docs/data-model.md` - draft data model and permissions notes.
- `docs/api.md` - API sketch and example payloads.
- `docs/roadmap.md` - phased delivery plan and milestones.
- `docs/plan.md` - near-term implementation checklist.
