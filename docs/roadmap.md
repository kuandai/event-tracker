# Roadmap

## Phase 0 - Scaffold (done)
- Basic Express server with in-memory data.
- Public event listing UI.
- Registration, login, and completion toggles.
- Admin quick-add endpoint and UI.

## Phase 1 - Persistence + Roles
- Add database (SQLite or Postgres).
- Replace SHA-256 with bcrypt.
- Admin roles stored per user.
- Event editing UI and audit trail.

## Phase 2 - Collaboration
- Organization profiles and multiple groups.
- Bulk import for events (CSV).
- Completion analytics for admins.

## Phase 3 - Notifications
- Due date reminders via email or SMS.
- Calendar exports (ICS) and integrations.
